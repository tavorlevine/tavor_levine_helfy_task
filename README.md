# Tavor Levine project:

### How to run the server ###
1. Open the project folder in VS code.
2. Open the terminal (Ctrl + ~) and install dependencies by running: "npm install".
3. After that run the following command: "node index.js".
4. Wait a few seconds until you see the message: "server is on - port 4000"
5. In a separate VS Code window, open the client project and run it.

### How to run the client ###
1. Open the client project folder in a new VS Code window.
2. Open the terminal (Ctrl + ~) and install the dependencies: "npm install"
3. Make sure that the server is up.
4. Run in the terminal the command: "npm run dev".
5. Open the provided local URL in your browser (usually http://localhost:5173).
